# Poorya Moradi's Personal Blog

This is a modern, responsive personal website and blog built with HTML, CSS, and JavaScript. It showcases my about section, blog posts, publications, experience, and contact information.

## Features
- Responsive design for mobile and desktop.
- Smooth scrolling navigation.
- Hero section for a welcoming introduction.
- Blog section for sharing articles and learnings.
- Easy to customize and expand.

## Getting Started
1. Clone the repository.
2. Open `index.html` in your browser.
3. Customize content in HTML, styles in CSS, and add functionality in JS as needed.
